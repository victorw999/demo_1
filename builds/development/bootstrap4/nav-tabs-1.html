ISSUE: Owl-carousel doesn't load on the first nav-tab pane?
SOLUTION: Because (ref) Bootstraps tabs are hidden initially
ISSUE: this is not a elegant solution. After applied this css, the preveious tab-pane was not hidden. The next tab-pane was just on top of the previous tab. I also tried with z-index. it won't fix the issue.
SOLUTION: it turned out we don't need the css after all. Just add active class to the first tab-pane
So I inserted below code to overirde the default bootstrap css .tab-content > .tab-pane{ display: block; height: 0; } .tab-content > .active{ height: auto; }
