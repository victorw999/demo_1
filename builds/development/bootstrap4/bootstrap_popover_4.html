Reference: buttons to control the popover manually.

Click on the following buttons to control the popover manually.

$(document).ready(function() { $(".show-popover").click(function() { $("#myPopover").popover('show'); }); $(".hide-popover").click(function() { $("#myPopover").popover('hide'); }); $(".toggle-popover").click(function() { $("#myPopover").popover('toggle'); }); $(".destroy-popover").click(function() { $("#myPopover").popover('destroy'); }); });
